🏠 Introducere — Ce face acest program?

Programul calculează costul de material și producție pentru oferte de piese tehnice (garnituri, profile, benzi adezive etc.), organizate pe articole. Fiecare articol are parametri geometrici, material, utilaj și coeficienți economici.

📑 Structura programului

TabRol
🧮 Program CalculIntroducere parametri piesă și calcul cost material per piesă
🧾 Rețetă PiesăCalcul detaliat cost producție: manoperă, utilaje, profit, transport
📋 CentralizatorRezumat toate articolele cu prețul final per piesă
📊 DashboardGrafice comparative între articole
⚙️ UtilajeParametri și costuri utilaje de producție
📚 ReferințeFormule, factori FM, tabele de referință
💡 Flux de lucru recomandat: Completați Program Calcul → ajustați Rețete Piesă → verificați Centralizator → Salvați oferta → Export Excel.

🎨 Codul de culori în tabele

  • Verde — câmp editabil direct
  • Galben — dropdown / selecție
  • Gri — valoare calculată automat (nu se editează)
  • Portocaliu — preț final

📌 Șablon Utilaje și FM

Butonul 📌 Salvează ca Șablon (disponibil în tabul Utilaje și în Referințe → FM) salvează un instantaneu complet al parametrilor curenți de utilaje și factorilor FM. Șablonul este stocat local în browser și poate fi actualizat oricând.

💡 Același buton 📌 Salvează ca Șablon apare și în tabul 📚 Referințe, secțiunea Factori FM.

📌 Șablon Utilaje și Referințe FM

Șablonul este un set de referință cu parametrii de utilaje și factori FM pe care îl definiți o singură dată și îl folosiți ca punct de pornire pentru ofertele noi.

Cum salvez un șablon?

  • Mergeți la tabul ⚙️ Utilaje și configurați parametrii
  • Mergeți la tabul 📚 Referințe și ajustați factorii FM
  • Apăsați 📌 Salvează ca Șablon (disponibil în ambele taburi)
  • Data și ora salvării sunt afișate lângă buton

Cum folosesc șablonul?

  • Apăsați ➕ Ofertă Nouă
  • Alegeți 📌 Din Șablon — se restaurează utilajele și FM-ul exact ca la momentul salvării
  • Alternativ, alegeți 📂 Din Proiect Existent pentru a prelua referințele dintr-un proiect anterior

Când actualizez șablonul?

Ori de câte ori modificați prețul energiei, adăugați utilaje noi sau ajustați factorii FM pentru lucrul curent. Șablonul nu afectează proiectele deja salvate.

⚠ Șablonul este stocat în browser (localStorage). Dacă ștergeți datele browserului, șablonul se pierde. Exportați BD regulat pentru backup complet.

💾 Salvare și Gestionare Oferte

📁 Selectarea folderului de salvare

La prima utilizare apăsați 📁 Selectează Folder și alegeți folderul unde se află Calcul_Oferta.html. Folderul este reținut automat. Dacă browserul este repornit, apăsați 🔓 Reconfirmă accesul din bannerul galben.

➕ Ofertă Nouă

La apăsarea butonului ➕ Ofertă Nouă apare un dialog cu două opțiuni:

  • 📌 Din Șablon — pornește cu parametrii de utilaje și FM salvați în șablon (recomandat pentru lucru standard)
  • 📂 Din Proiect Existent — alege un proiect salvat și preia utilajele și FM-ul lui; articolele nu se copiază
💡 Salvați mai întâi un șablon din tabul ⚙️ Utilaje → 📌 Salvează ca Șablon înainte de a crea oferte noi.

💾 Salvarea unei oferte

  • Completați Denumire Ofertă și Nr. Proiect
  • Apăsați 💾 Salvează Ofertă sau Ctrl+S
  • La fiecare salvare se creează o versiune nouă (v1, v2, v3...)
  • Fiecare versiune salvează și utilajele + FM-ul exact din acel moment

📂 Încărcarea unei oferte

  • Apăsați 📂 Încarcă Proiect
  • Filtrați după Nr. Proiect, Versiune sau Dată
  • Click pe versiunea dorită → se încarcă complet

🗄️ Baza de date

Toate ofertele sunt stocate local în browser (localStorage). Folosiți ⬆ Export BD (JSON) pentru backup și 📥 Import BD pentru restaurare.

⚠ Ștergerea datelor de browser șterge și baza de date! Faceți backup regulat cu Export BD.

📥 Import / Export

📊 Export Excel

Butonul 📊 Export Excel generează un fișier .xlsx cu 4 sheet-uri: Program Calcul, Rețete, Centralizator, Utilaje. Fișierul se salvează automat în folderul selectat.

📥 Import Ofertă Veche (format PRJM Export)

Permite importul articolelor din fișiere xlsx în formatul vechi de export.

  • Apăsați 📥 Import Ofertă Veche
  • Selectați fișierul xlsx
  • Verificați previzualizarea articolelor detectate
  • Alegeți Adaugă la existente sau Înlocuiește tot
  • Apăsați ✅ Importă
💡 Câmpurile calculate (Cost/p, Preț/p etc.) se recalculează automat după import.

⬆ Export BD (JSON)

Exportă întreaga bază de date cu toate ofertele și versiunile ca fișier JSON. Folosiți pentru backup sau transfer pe alt calculator.

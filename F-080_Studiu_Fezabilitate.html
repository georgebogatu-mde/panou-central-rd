❓ Ghid de utilizare — F-080 Studiu de Fezabilitate

📋 Prezentare generală

Aplicația F-080 Studiu de Fezabilitate permite crearea, editarea, salvarea și exportul studiilor de fezabilitate conform standardului intern MDE Converting.

Bara de instrumente (Toolbar)

  • 💾 Salvează — Salvează versiunea curentă în baza de date locală (crează o versiune nouă)
  • 📂 Încarcă — Deschide lista studiilor salvate pentru a le edita
  • 📊 Export Excel — Generează fișierul .xlsx cu formatul oficial F-080
  • ⬆ Export BD — Exportă întreaga bază de date ca fișier JSON (backup)
  • 📥 Import BD — Importă o bază de date din fișier JSON
  • 📊 Import Excel — Re-importă date dintr-un Excel exportat anterior
  • ＋ Nou — Creează un studiu nou (gol)
💡 Indicator de stare: Badge-ul colorat din bara de titlu arată versiunea curentă.
• v3 — salvat  • v3 * — modificat, nesalvat  • NOU * — nesalvat niciodată

💾 Salvare & Gestionare Versiuni

Cum funcționează versiunile

La fiecare apăsare pe Salvează, se crează o versiune nouă (v1 → v2 → v3...). Versiunile anterioare rămân disponibile în baza de date — nicio versiune nu este suprascrisă.

Salvare rapidă

Folosiți scurtătura Ctrl + S pentru a salva fără a folosi mouse-ul.

Ștergerea unei versiuni

1
Apăsați 📂 Încarcă pentru a deschide lista studiilor
2
Găsiți versiunea dorită și apăsați butonul 🗑
3
Confirmați ștergerea în fereastra de dialog
💡 Resetare la v1: Dacă ștergeți versiunea curentă (de ex. v3), datele completate rămân intacte în formular. La următoarea apăsare pe Salvează, studiul va fi salvat ca v1 cu un ID nou.

Căutare & filtrare în lista de studii

  • Filtru PRJ / Denumire — caută după numărul de proiect sau denumire
  • Filtru Versiune — afișează doar o anumită versiune
  • Filtru Dată — filtrează după data întocmirii
  • Butonul ✕ Resetează șterge toate filtrele active
⚠ Atenție: Baza de date este stocată în localStorage al browserului. Dacă ștergeți datele browserului sau folosiți un alt browser, veți pierde studiile. Folosiți Export BD pentru backup periodic!

📊 Export & Import Excel

Export Excel

Apăsați 📊 Export Excel pentru a genera fișierul .xlsx în formatul oficial F-080 cu logo MDE în header-ul de pagină.

Numele fișierului generat: F-080_SF_[PRJ]_[Denumire20]_v[N]_[Data].xlsx

  • Dacă ați setat un folder de lucru, fișierul se salvează direct în el
  • Altfel, se descarcă automat în folderul Downloads

Import Excel

Permite re-încărcarea datelor dintr-un fișier Excel exportat anterior (util dacă baza de date JSON a fost pierdută).

1
Apăsați 📊 Import Excel din toolbar sau din fereastra Încarcă
2
Selectați fișierul .xlsx exportat anterior din această aplicație
3
Datele sunt importate automat în toate secțiunile formularului
💡 Ce se importă: Customer Requirements, Technical Feasibility, Capacity/Suppliers, Lessons Learned, Decision, Edition History, numărul de proiect și denumirea.
⚠ Notă: Câmpurile Dată întocmirii și Versiune nu sunt stocate în Excel — vor apărea goale după import și trebuie completate manual. Apăsați Salvează după import pentru a înregistra studiul în baza de date.

📁 Folder de Lucru

Folderul de lucru este un director de pe calculatorul dvs. unde aplicația salvează automat fișierele Excel exportate și baza de date JSON.

Configurare folder

1
Apăsați butonul 📁 din bara de sus (zona FOLDER LUCRU)
2
Selectați folderul dorit din dialogul sistemului de operare
3
Browserul va cere confirmare de acces — apăsați Permiteți

Comportament cu folder activ

  • Export Excel → salvat direct în folder (nu mai descarcă în Downloads)
  • Salvează → baza de date JSON sincronizată automat în folder
  • Export BD → fișierul JSON salvat în folder
💡 Persistență: Folderul ales este reținut între sesiuni. La redeschiderea aplicației, veți fi întrebat dacă confirmați accesul la același folder.
💡 Backup automat: Dacă setați folderul la Documents → Claude → Projects → Program Studiu de Fezabilitate, toate fișierele sunt organizate automat alături de aplicație.

Dezactivare folder

Apăsați butonul ✕ din zona FOLDER LUCRU pentru a renunța la folderul selectat.

📝 Secțiunile Formularului

1. Customer Requirements

Cerințele clientului — specificații, caracteristici speciale, status fezabilitate per cerință.

  • Status: OK / RISK / NOK — colorat automat (verde / portocaliu / roșu)
  • Validation Required: YES / NO

2. Technical Feasibility

Capabilitate tehnică per proces de fabricație (Die Cutting, Slitting, Lamination, etc.).

  • Machine Capability: YES / NO
  • Cpk Target: valoare numerică sau text (ex. ≥1.33)

3. Capacity / Suppliers

Capacitate de producție și evaluare furnizori de materiale.

  • EYV: cerere anuală estimată (buc/an)
  • Approved Material / Supplier Evaluation: YES / NO / PARTIAL / INTERIM

4. Lessons Learned

Lecții din proiecte anterioare aplicabile acestui studiu.

5. Decision

Decizia finală de fezabilitate cu semnături de aprobare.

  • GO — fezabil, colorat verde
  • CONDITIONAL — fezabil cu condiții, colorat galben
  • NO GO — nefezabil, colorat roșu

6. Edition History

Istoricul modificărilor documentului. Ediția se numerotează automat. Apăsați ＋ Ediție nouă pentru a adăuga o intrare.

Adăugare / Ștergere rânduri

Fiecare secțiune are un buton ＋ Adaugă rând. La al doilea click (când există deja un rând), apare un dialog de copiere care permite duplicarea unui rând existent sau adăugarea unui rând gol.

Rândurile se șterg cu butonul ✕ din coloana din dreapta.

⌨ Scurtături de Tastatură & Sfaturi

Scurtături tastatură

  • Ctrl + S — Salvează studiul curent (crează versiune nouă)

Flux de lucru recomandat

1
Completați câmpurile PROIECT și Denumire din bara de informații a documentului
2
Completați toate secțiunile formularului (Customer, Technical, Capacity, etc.)
3
Apăsați Ctrl+S sau 💾 Salvează — se crează v1
4
Apăsați 📊 Export Excel pentru a genera documentul oficial
5
La actualizare: modificați datele, adăugați o ediție nouă, salvați → se crează v2, v3...
6
Folosiți periodic ⬆ Export BD pentru backup-ul bazei de date

Sfaturi utile

  • Câmpul Denumire este folosit și în numele fișierului Excel exportat (primele 20 de caractere)
  • Dacă pierdeți baza de date JSON, puteți re-importa datele direct din fișierele Excel exportate (📊 Import Excel)
  • Setați un folder de lucru pentru a sincroniza automat toate fișierele într-un singur loc
  • Versiunile vechi nu se șterg automat — faceți curățenie periodic din fereastra Încarcă
💡 Recuperare date: Chiar dacă browserul este închis accidental, studiul curent este salvat automat în sessionStorage și se recuperează la redeschidere.
